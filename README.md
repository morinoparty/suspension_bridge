# Get started
This is a minecraft proxy.
It is implemented by golang and [gate](https://gate.minekube.com/)

## Getting Started
### Go
```bash 
go run main.go
```

or

```bash
go build -o suspension_bridge
./suspension_bridge
```

### Docker
```bash
docker build -t suspension_bridge .
docker run -p 25565:25565 suspension_bridge
```
